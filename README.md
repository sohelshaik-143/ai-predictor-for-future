# AI-Powered Daily Worker Financial Protection System

Scaffold for a production-ready system that tracks daily income for informal workers, predicts income instability, computes financial risk, provides AI-generated savings and migration advice, and offers a 3-year roadmap.

Services:
- `backend` — Spring Boot REST API (JWT, MongoDB)
- `ml-service` — FastAPI ML microservice (prediction & risk)
- `frontend` — Static SPA (Tailwind, Vanilla JS, Web Speech API)
- `MongoDB` — mongodb as database 

Quick start (local, Docker): see `docker-compose.yml` and `docs/DEPLOYMENT.md`.
